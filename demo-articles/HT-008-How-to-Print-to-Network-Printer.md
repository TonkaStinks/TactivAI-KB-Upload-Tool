# How to Print to Network Printer

## When to Use This Article
Use this guide when you need to set up a network printer on your computer or troubleshoot printing issues.

## Prerequisites
- Computer connected to the office network (WiFi or ethernet)
- If working remotely, must be connected to VPN
- Know the printer name or location

## Step-by-Step Instructions

### Step 1: Open Printer Settings
**On Windows:**
1. Press Windows key + I to open Settings
2. Click "Bluetooth & devices" > "Printers & scanners"
3. Click "Add device"

**On Mac:**
1. Click Apple menu > System Preferences
2. Click "Printers & Scanners"
3. Click the + button to add a printer

### Step 2: Find Your Printer
The computer will search for available printers. Look for:
- **Floor 1 Printer** (HP LaserJet in lobby)
- **Floor 2 Color** (Canon color printer by kitchen)
- **Floor 3 Main** (Xerox multifunction)

Click on the printer you want to add.

### Step 3: Install the Printer
1. Click "Add device" or "Add"
2. Windows/Mac may download drivers automatically
3. Wait for "Ready" status

### Step 4: Set as Default (Optional)
To make this your default printer:
1. Click on the printer in your list
2. Click "Set as default" or check "Default printer"

### Step 5: Test Print
1. Open any document
2. Press Ctrl+P (Windows) or Cmd+P (Mac)
3. Select your new printer
4. Click "Print"
5. Walk to the printer to collect your document

## Printing Tips

### Print Double-Sided
1. In the print dialog, look for "Print on both sides"
2. Select "Flip on long edge" for normal documents

### Print in Black & White
1. Find "Color" settings in print dialog
2. Select "Black & White" or "Grayscale"
3. Saves color ink for when you really need it

### Secure Print (Confidential Documents)
1. Select "Secure Print" or "PIN Print" in options
2. Enter a 4-digit PIN
3. At the printer, enter your PIN to release the job

## Common Issues

### Issue 1: Printer Not Found
**Symptoms:** Printer doesn't appear in search
**Solution:**
- Make sure you're on the company network (not guest WiFi)
- If remote, connect to VPN first
- Ask a coworker for the exact printer name

### Issue 2: Print Job Stuck in Queue
**Symptoms:** Document shows "Printing" but never comes out
**Solution:**
- Open printer queue and cancel the stuck job
- Turn printer off and on
- Try printing again

### Issue 3: Prints Wrong Size or Cut Off
**Symptoms:** Document doesn't fit on page
**Solution:**
- Check print settings for "Fit to page" option
- Verify paper size matches your document (Letter vs A4)
- Check page orientation (portrait vs landscape)

### Issue 4: Low Quality or Faded Prints
**Symptoms:** Text is light or streaky
**Solution:**
- The printer may need new toner - check the display
- Try a different printer
- Report to IT for toner replacement

## Escalation Triggers
- Printer shows error code on display
- Paper jam that won't clear
- Printer offline for more than 30 minutes
- Need to print confidential/secure documents

## Success Criteria
- Printer appears in your printer list
- Test page prints successfully
- Can print from any application
